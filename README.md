# Agenda - Sistema de Gerenciamento de Contatos

Este sistema foi desenvolvido utilizando o framework Django, com o objetivo de criar uma agenda de contatos para gerenciar e organizar contatos de forma eficiente.

# Features:
- Cadastro de contatos com nome, sobrenome, telefone, e-mail e descrição
- listagem de contatos
- busca de contatos por nome ou telefone
- edição de contatos
- exclusão de contatos

# Tecnologias utilizadas:
- Django
- HTML
- CSS
- JavaScript
- Banco de dados SQLite
